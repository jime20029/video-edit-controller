## インストール

### 前提条件

- Python 3.9.6
- pip

### 環境構築手順

#### 仮想環境のアクティブ化

#### clone 後にプロジェクトディレクトリで以下を実行（仮想環境の設定）

```
python3 -m venv .venv
```

#### 仮想環境のアクティブ化

```
source .venv/bin/activate
```

#### 各種ライブラリのインストール

```
pip install cryptography
```
